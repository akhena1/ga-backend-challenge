# 🏆 Desafio Backend - NestJS + GraphQL + TypeORM

## Pré-requisitos para executar a aplicação:
 - Node.js
 - Docker

## Instalação das dependências

```bash
$ npm install
```

## Executar a aplicação em modo de desenvolvimento

```bash
$ docker-compose up
```
## Para executar os testes

```bash
# Unitários
$ npm run test
```
```bash
# e2e
$ npm run test:e2e
```
